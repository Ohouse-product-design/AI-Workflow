솔루션 타입 가이드

각 솔루션 타입의 특성과 적합한 사용 케이스입니다.

Claude Skill
슬래시 커맨드로 즉시 실행
반복적 단일 작업. 프롬프트 템플릿 기반. 개인이 직접 호출.

예: 체크리스트 생성, 데이터 조회, UT 시나리오
Claude Project
시스템 프롬프트 + 문서 컨텍스트
전용 환경을 세팅하여 특정 도메인에 최적화된 응답 생성. ODS 등 대규모 컨텍스트 포함.

예: Design-draft (ODS 컨텍스트 포함)
문서/템플릿
코드 없이 Notion, MD, 체크리스트로 정리
가장 많고 가장 중요한 기초 작업. 패턴 정리, 가이드 작성, 수동 큐레이션 등.

예: 패턴 정리, 정책 가이드, 온보딩 체크리스트
기존 도구 활용
이미 있는 Figma Plugin/Dev Mode 등 활용
새로 만들지 않고 기존 도구를 그대로 활용. Design Lint, Content Reel 등.

예: Design Lint, Figma Dev Mode, Content Reel
MCP / Slack
외부 시스템 연동
Athena, Redash MCP 또는 Slack Bot을 통해 외부 데이터/워크플로우에 연결.

예: 데이터 조회, ODS QnA 봇, 크리틱 봇
