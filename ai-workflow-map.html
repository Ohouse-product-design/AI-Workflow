Figma · UI 생성 Agent · 패턴 분석
P1
Responsive-converter
생성된 UI의 PC/모바일/태블릿 반응형 레이아웃 자동 변환. 디자인 시스템 브레이크포인트 기준 적용.
Figma Plugin
DS AI-Ready — LLM이 디자인을 이해하기 위한 Layer 구축
P0
DS-logic-builder
Logic Layer
컴포넌트 속성(Variant/Size/State) 정규화, 네이밍 린팅(Category/Component/Variant/State 규칙), Auto Layout · 토큰 매핑 검증. 레이어명 "Frame 124" → "Button Container"로 정비.

LLM이 컴포넌트의 구조와 동작 방식을 파악할 수 있게 하는 기반
Figma Plugin · 린팅 Skill · 속성 정규화
P0
DS-knowledge-base
Knowledge Layer
컴포넌트별 사용 규칙(Primary 버튼은 섹션당 1개), 제약 조건(최소 터치 44px, 명암비 4.5:1), 접근성 기준을 기계 읽기 가능한 구조화 문서로 생성. 디자인 의도와 맥락까지 포함.

LLM이 "왜 이렇게 쓰는지"를 이해하고 올바른 UI를 생성하게 하는 지식
Skill · 규칙 문서화 Agent · 지식 추출
P1
DS-code-bridge
Design ↔ Code 연결
토큰을 JSON/코드로 내보내기, Figma 컴포넌트 ↔ 프론트엔드 코드 매핑 동기화. 디자인과 코드가 동일한 구조를 공유해 LLM이 양쪽을 오갈 수 있게 연결.

디자인 변경이 코드에 자동 반영되는 파이프라인
MCP · Token Sync Figma · Code Connect

Platform Layer

전체 워크플로우를 지탱하는 공통 인프라
P0
Skill-hub
팀 공통 스킬 저장소. 스킬/에이전트를 등록 · 검색 · 버전 관리. 누가 만들었는지, 어떤 프롬프트 · MCP 조합인지, 사용법 가이드까지 한 곳에서 관리.

팀원이 만든 스킬을 다른 팀원이 바로 /skill로 쓸 수 있는 구조
Git Repo · 스킬 저장소 Skill · 레지스트리
P1
Skill-quality-gate
공유 스킬의 품질 관리. 프롬프트 테스트 · 결과 일관성 검증 · 사용 빈도 트래킹. 잘 쓰이는 스킬은 프로모션, 안 쓰이는 스킬은 정리.

스킬이 쌓이기만 하고 관리 안 되는 문제 방지
Agent · 품질 검증 Skill · 테스트 러너
P0
Insight-archive
팀의 실험 결과, 지표 하이라이트, 핵심 인사이트, 도메인 지식을 구조화해서 축적. 신규 입사자/조직 이동 시 "왜 이런 결정을 했는지"를 AI가 맥락과 함께 답변. Notion · Slack 히스토리에서 자동 수집 → 검색 가능한 지식 베이스로 정리.

3년차가 1주일 만에 가질 수 있는 도메인 맥락
Agent · 지식 수집 MCP · Notion MCP · Slack
P1
MCP-connector-hub

솔루션 타입 가이드

각 솔루션 타입의 특성과 적합한 사용 케이스입니다.

Claude Skill
반복적인 단일 작업
슬래시 커맨드(/query, /critique)로 즉시 실행. 프롬프트 + 도구 조합. 개인이 직접 호출.

예: 데이터 조회, UX 라이팅 검수, 스펙 문서 생성
Claude Agent
복잡한 멀티스텝 작업
여러 도구를 자율적으로 조합해 수행. 수집 → 분석 → 정리 등 연쇄 작업. 백그라운드 실행 가능.

예: VOC 수집+분류, 피드백 추적, 경쟁사 리서치
Figma Plugin
디자인 도구 내 자동화
Figma 안에서 직접 실행. 컴포넌트 생성/검증/변환 등 시각적 작업. MCP로 연동 가능.

예: 상태별 UI 생성, 핸드오프 어노테이션, QA 비교
MCP Plugin
외부 시스템 연결
Athena, Redash, Notion, Google Drive 등 외부 데이터 소스 연결. Skill/Agent의 백엔드 역할.

예: 데이터 조회, Slack 메시지 읽기, Notion DB 접근
Slack Bot
팀 채널 기반 자동화
Slack 채널에서 자동 게시/알림. 팀 전체가 수동 호출 없이 혜택. 스케줄 기반 실행.

예: 크리틱 자동 게시, ODS QnA 응답, 공유 알림
