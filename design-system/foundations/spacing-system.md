# 간격 시스템 가이드

## 4px 기반 스케일

모든 간격은 **4px 배수**입니다. 임의의 값(예: 7px, 15px, 30px) 사용을 금지합니다.

```
4 → 8 → 12 → 16 → 20 → 24 → 32 → 40 → 48 → 64 → 80 → 96
```

## 간격 유형

### Inset (내부 여백 / padding)
컨테이너 내부 패딩에 사용합니다.

| 토큰 | 값 | 사용처 |
|---|---|---|
| `inset.xs` | 4px | 칩, 뱃지 내부 |
| `inset.sm` | 8px | 작은 버튼, 태그 |
| `inset.md` | 16px | 카드, 인풋, 일반 버튼 |
| `inset.lg` | 24px | 모달, 바텀시트 내부 |
| `inset.xl` | 32px | 페이지 섹션 |

### Stack (수직 간격)
세로로 쌓이는 요소 간의 간격입니다.

| 토큰 | 값 | 사용처 |
|---|---|---|
| `stack.xs` | 4px | 라벨과 인풋 사이 |
| `stack.sm` | 8px | 같은 그룹 내 요소 |
| `stack.md` | 16px | 요소 그룹 간 |
| `stack.lg` | 24px | 섹션 내 블록 간 |
| `stack.xl` | 40px | 섹션 간 |

### Inline (수평 간격)
가로로 나열되는 요소 간의 간격입니다.

| 토큰 | 값 | 사용처 |
|---|---|---|
| `inline.xs` | 4px | 아이콘과 텍스트 |
| `inline.sm` | 8px | 칩 간격 |
| `inline.md` | 12px | 버튼 그룹 |
| `inline.lg` | 16px | 카드 간격 |
| `inline.xl` | 24px | 컬럼 간격 |

## 레이아웃 간격

### 페이지 마진
| 디바이스 | 마진 |
|---|---|
| Mobile (< 768px) | 16px |
| Tablet (768~1023px) | 24px |
| Desktop (>= 1024px) | 40px |

### 섹션 간격
| 디바이스 | 간격 |
|---|---|
| Mobile | 32px |
| Tablet | 40px |
| Desktop | 48px |

### 그리드
| 디바이스 | 컬럼 | 거터 |
|---|---|---|
| Mobile | 4 | 16px |
| Tablet | 8 | 20px |
| Desktop | 12 | 24px |

최대 콘텐츠 너비: **1200px**

## 실전 예시

### 카드 내부 레이아웃
```
┌─────────────── Card ───────────────┐
│  ← inset.md (16px) →              │
│  ┌──────── Image ────────┐        │
│  └───────────────────────┘        │
│  ↕ stack.sm (8px)                 │
│  Title (Heading 3)                │
│  ↕ stack.xs (4px)                 │
│  Description (Body 2)             │
│  ↕ stack.md (16px)                │
│  ┌─ Button ─┐                     │
│  └──────────┘                     │
└────────────────────────────────────┘
```

### 리스트 아이템
```
┌─────────── List Item ──────────────┐
│  ← inset.md (16px) →              │
│  [Icon] ← inline.sm (8px) → Text  │
│  ↕ border (1px)                    │
└────────────────────────────────────┘
↕ stack.xs (0~4px, 보더로 구분 시)
```
