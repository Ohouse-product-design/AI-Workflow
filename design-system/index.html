AI Agent Guide

Claude Code / Claude Projects에서 ODS를 활용하기 위한 가이드입니다.

컨텍스트 파일

파일용도
DESIGN.md9섹션 메인 디자인 시스템 문서 (Claude Projects Knowledge에 업로드)
ai-context/system-prompt.md시스템 프롬프트 템플릿 (PRD → HTML 생성 워크플로우)
ai-context/component-rules.md컴포넌트별 Do's & Don'ts 규칙
ai-context/naming-convention.md네이밍 컨벤션 (Figma 레이어, CSS 클래스, 토큰)

사용 예시

Step 1: DESIGN.md를 Claude Projects Knowledge에 업로드 Step 2: system-prompt.md 내용을 System Prompt에 붙여넣기 Step 3: PRD 입력 → HTML 자동 생성 (A/B/C 3안) Step 4: 검증 루프 (구조 → 시각 → 반응형)
